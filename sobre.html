Guia de Uso do Sistema

Aprenda de forma simples como consultar, analisar e registrar dados com o S&S.

O que é o S&S

O S&S é um sistema que ajuda você a consultar, capturar e analisar informações a partir de fichas e imagens. Ele usa IA para extrair campos importantes (como material, local, data e exatidão) e permite registrar os resultados com rastreabilidade para auditorias futuras.

Como funciona

1) Acesso: Navegue até a aba "Consulta" (o núcleo operacional do sistema).
2) Captura: Utilize a câmera em tempo real para capturar a foto do formulário.
3) Processamento e Validação: O OCR extrai os dados e preenche os campos automaticamente. Nesta etapa, o usuário deve conferir os campos lidos.
4) Registro: O sistema grava os campos lidos no banco de dados.
5) Visualização: Os dados lidos e tratados devem ser visualizados no Power BI.
